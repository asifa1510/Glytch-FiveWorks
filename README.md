# GLYTCH Neuro-Assist System  
**World's First Fully Offline Wrist-Twist Communication + Fall Alert + Medicine OCR System** ⚡

**Novelty & Impact**: The only open-source, zero-internet-required neuro-assistive platform that combines intentional wrist-twist communication (YES/NO), multi-sensor fall detection, and standalone ESP32-CAM medicine OCR — all working offline. Designed, built & tested with real ALS & Parkinson's patients in India, 2025.

### Groundbreaking Features (All Offline-Capable)
- Clockwise/counterclockwise wrist twist → YES/NO (100% success in patient testing)
- Multi-factor fall detection (MPU6050 + piezo) → instant SMS/Call + GPS
- ESP32-CAM standalone medicine scanner (OCR → "Paracetamol|500mg")
- NeuroMind real-time tremor & emotional state tracking
- AI Assistive Glasses prototype (on-device object detection)

**Hardware**: ESP32 + MPU6050 + Piezo + LiPo wristband | ESP32-CAM | Android App (Kotlin)
